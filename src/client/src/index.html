>WebProgramming Project

wfc2d-multiplayer-techdemo

This tech demo has been developed with the main purpose of studing:

  • Wave Function Collapse algorithm for map generation
  • Socket.IO technology for a multiplayer gameplay

This little project is a pretty way to make the two things work together.

>how to play

  1. Press play game

  2. Insert your nickname and press Go!

  3. Explore the map and meet other online users

**use WASD to move and press R to respawn if you get stuck ^-^ **
